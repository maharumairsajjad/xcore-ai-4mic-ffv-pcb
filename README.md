# xcore.ai 4-Mic FFV PCB Design

## Overview
This repository contains the complete PCB design for a **4-Microphone Far-Field Voice (FFV) system** based on **xcore.ai**, optimized for AI-driven voice recognition, noise suppression, and embedded audio applications.

The board is designed for **far-field speech capture**, ensuring high signal integrity, low noise, and efficient power management.

## Key Features
- 4-Microphone Array for far-field voice capture
- Optimized analog & digital audio routing
- Low-noise power architecture
- AI-ready audio processing platform
- Designed for smart assistants and IoT devices

## Hardware Highlights
- Multi-layer PCB with controlled impedance routing
- Separate analog and digital ground regions
- EMI-aware microphone placement
- Robust clock and power decoupling strategy
